##### 5. 检查配置
- 在检查配置前，要先进行环境变量。
- 我的电脑--右键--属性--高级系统设置--环境变量。
![13.1](img/13.1.png) 
- 用户变量--新建：JAVA_HOME；变量值：如图。
- 编辑--变量名"Path"--在原变量值最后添加%JAVA_HOME%\bin。
![13.2](img/13.2.png)

- 打开cmd,输入Java，出现以下结果，即证明没存在问题。
![14](img/14.png)
- 输入javac,若出现以下结果，这证明环境变量没问题。
![15](img/15.png)
